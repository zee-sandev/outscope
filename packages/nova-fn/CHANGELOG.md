# @outscope/nova-fn

## 2.1.0

### Minor Changes

- Add `defineHandle(access)` typed policy helpers and composable access policies with `kind`, `uses`, and access middleware composition.

## 2.0.0

### Major Changes

- Introduce Nova 2.0 DX vocabulary: `routes`, `access`, and `handlers`.
- Add `defineAccess`, `defineHandlers`, and `handle.public/auth/permission`.
- Replace functional operation maps with handler maps.
- Add access metadata to handler context as `ctx.access`.

## 1.0.0

### Major Changes

- Release major version for all packages.
